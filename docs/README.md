# docs の構成

このディレクトリには、ハンズオンの各ステップに対応する資料や関連ファイルが含まれています。各ディレクトリの構成と内容について以下に説明します。

## ディレクトリ構成

- **step1, step2, ...**  
  各ステップのハンズオン内容が記述されています。ディレクトリ名はハンズオンの順序を表しています。

  - **README.md**  
    ハンズオンの具体的な手順や説明が記載されています。

  - **img/**  
    ハンズオンの手順を補足するためのスクリーンショットが保存されています。

  - **src/**  
    そのステップで作成するソースコードの完成形が保存されています。コードの参考や動作確認に使用できます。

## 使用方法

1. 各ステップのディレクトリに移動し、`README.md` を参照してください。
2. 必要に応じて `img/` 内の画像を確認しながら進めてください。
3. 実装に迷った場合は、`src/` 内の完成形コードを参考にしてください。

## 注意事項

- 各ステップは順番に進めることを想定しています。前のステップを完了してから次のステップに進んでください。
- `src/` 内のコードは完成形のため、直接コピーするのではなく、理解しながら実装することを推奨します。
